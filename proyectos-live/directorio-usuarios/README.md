# Ejercicio de entrevista, directorio de usuarios

Podemos encontrar tres carpetas [caso-0](./caso-0), [caso-1](./caso-1) y [caso-2](./caso-2) que contienen diferentes desafíos para una misma aplicación, un directorio de usuarios.

Cada caso se vuelve un poco más difícil:

* **Caso 0** apunta a perfiles trainee / junior (listar datos y estados de carga).
* **Caso 1** apunta a perfiles junior / semi-senior (búsqueda y filtros).
* **Caso 2** apunta a perfiles semi-senior / senior (formularios, validación, orden y persistencia).

Cada caso está pensado para resolverse en aproximadamente 30 minutos.
