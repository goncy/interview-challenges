# Ejercicios de entrevista

## Nivel 1
- [ ] No se muestran los usuarios al cargar la aplicación
- [ ] El buscador no filtra correctamente los usuarios

## Nivel 2
- [ ] Mostrar un texto de "Cargando..." hasta que obtengamos los datos
- [ ] Mostrar la cantidad total de usuarios en el encabezado

## Nivel 3
- [ ] Agregar un filtro por estado (activo / inactivo / todos)
- [ ] Mostrar un contador "X de Y usuarios" según los filtros aplicados

## Extra
- [ ] Mostrar un mensaje "No hay usuarios" cuando la lista esté vacía
- [ ] Mostrar un mensaje "Sin resultados" cuando la búsqueda no encuentra coincidencias
